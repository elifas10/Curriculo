4ALL Tests

Nome: Elifas Martins Zaire Andrade

Curso: Turma 4 - Programa de Migração em Testes de Software.

Recursos do Projeto:

    • Planejamento e técnicas de Testes: Utilização de metodologias eficazes de planejamento e técnicas de testes para garantir a qualidade do software.
      
    • Testes de API: Realização de testes em interfaces de programação de aplicativos para assegurar o correto funcionamento e integração dos serviços.
      
    • Testes Automatizados com Cypress (Web e API): Implementação de testes automatizados utilizando o framework Cypress, tanto para testes de interface web quanto para testes de APIs.
      
    • Metodologias Ágeis e Filosofia Lean: Aplicação de metodologias ágeis e princípios da filosofia Lean para garantir a entrega rápida e eficiente de software de alta qualidade.
