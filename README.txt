Estrutura de Diretórios
css\ - Arquivos de folhas de estilo CSS para o modelo.
images\ - Imagens usadas no modelo.
scripts\ - JavaScript usado no modelo.
styles\ - Arquivos de origem SCSS para folhas de estilo CSS. Disponível apenas nas versões compradas.
favicon.ico - Ícone do site fornecido para o modelo.
index.html - Página HTML principal para abrir o modelo no navegador.
.html - Páginas HTML adicionais (se houver).
LICENSE-.txt - Arquivo de licença conforme a variante baixada.

Como Editar o Modelo
Edição de Conteúdo no Modelo
Para editar o conteúdo, abra os arquivos *.html correspondentes e abra-os
em um editor como o VSCode ou Notepad++.
Procure o conteúdo que deseja alterar e edite-o para o texto desejado.
Salve o arquivo e reabra o arquivo HTML no navegador para ver as alterações.

Edição de Estilos do Modelo
O estilo principal está presente em css\main.css que você pode editar.
Você deve saber como editar arquivos CSS para fazer isso.
Se você comprou alguma das versões premium, também receberá
arquivos de origem SCSS que geram todos os estilos do modelo.
Você pode usar esses arquivos SCSS para fazer suas alterações e regenerar CSS.

Alterando Imagens
Você pode alterar as imagens dentro da pasta images.
Certifique-se de nomear o arquivo da mesma forma que o arquivo original.
Se desejar alterar o nome do arquivo de imagem, você deve
também fazer a alteração correspondente no arquivo HTML que se refere a essa imagem.

Como usar o Modelo
Faça o upload dos conteúdos para o seu servidor da web.
index.html deve estar na pasta principal (raiz) do seu servidor da web.
Não altere o nome das pastas ou arquivos do modelo a menos que saiba
o que está fazendo e esteja alterando os nomes nos arquivos HTML também.
Para hospedagem na web, você pode ir para https://templateflip.com/go/hosting para fácil upload de arquivos FTP.
